this is a small batch and gdb command file which allows automatic creation of backtrace log

3 simple steps:
- you have to have installed gdb
- copy the 2 files into the kvirc directory which contains the binaries with debug informations
- run create_debuglog.bat

after that that the directory willl contain 2 more files:
- debug_stdout.txt contains backtrace and other informations
- debug_stderr.txt contains debug output
